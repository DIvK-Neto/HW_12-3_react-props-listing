# Функциональность

## Основные возможности

- **Отображение списка предложений** из файла `etsy.json`.
- **Компонент `Listing`**:
  - Принимает массив товаров через пропс `items`.
  - Фильтрует удалённые позиции (`state !== 'removed'`).
- **Форматирование данных**:
  - Обрезка заголовка до 50 символов с добавлением `…`.
  - Преобразование цены в зависимости от валюты.
  - Определение класса остатка (`stock-low`, `stock-medium`, `stock-high`).
- **Адаптивная сетка** – количество колонок меняется в зависимости от ширины экрана.

## Вспомогательные функции

- `getCurrencySymbol(code)` – возвращает символ валюты или код + пробел.
- `getStockLevelClass(quantity)` – возвращает строку с CSS-классами.
- `truncateTitle(title, maxLength)` – обрезает строку до указанной длины.

## Стилизация

Используется CSS-файл `Listing.css`, повторяющий стили из примера. Карточки имеют тень и анимацию при наведении.
