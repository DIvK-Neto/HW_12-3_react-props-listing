# Используемые технологии

- **React 19** – библиотека для построения UI.
- **TypeScript** – типизация пропсов и данных.
- **Vite** – сборщик и сервер разработки.
- **CSS** – стилизация компонента (включая адаптивную сетку).

## Инструменты качества кода

- **ESLint** – проверка синтаксиса и стиля.
- **Prettier** – форматирование кода.
