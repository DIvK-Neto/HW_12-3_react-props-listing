# Домашнее задание к занятию «Props и Typescript»

## Список предложений

Необходимо вывести список предложений каталога Etsy.com, используя библиотеку React. После загрузки данных и отрисовки список должен выглядеть как на скриншоте.

### Данные списка предложений

Данные в формате JSON находятся в файле `data/etsy.json`. Это массив объектов. Используются поля:

- `listing_id` – уникальный идентификатор, число;
- `url` – ссылка на предложение, строка;
- `MainImage.url_570xN` – адрес главного изображения, строка;
- `title` – название предложения, строка;
- `currency_code` – код валюты, строка;
- `price` – цена, строка;
- `quantity` – доступное количество, число.

### Описание компонента

Создать компонент `Listing`, который принимает пропс `items` (массив объектов, по умолчанию пустой массив). Компонент формирует HTML-код карточки товара:


```html
<div class="product-card">
  <img src="..." alt="..." class="product-image">
  <div class="product-info">
    <h3 class="product-title">...</h3>
    <div class="price-container">
      <div class="product-price">...</div>
      <span class="stock-badge stock-...">... left</span>
    </div>
  </div>
</div>
```

### Правила отображения

- Название обрезается до 50 символов с добавлением `…`.
- Цена форматируется:
  - USD → `$50.00`
  - EUR → `€50.00`
  - GBP → `£50.00`
  - остальные → `CAD 50.00`
- Класс остатка зависит от количества:
  - `stock-low` – ≤10
  - `stock-medium` – ≤20
  - `stock-high` – >20

### Реализация

Отобразить данные с помощью компонента `Listing`. Стили и разметку можно взять из папки `markup`.
